# 💖 Romantic Proposal Website

A beautiful, interactive website to propose to your special someone!

## 🎯 How to Customize

### Change Her Name
Open `script.js` and modify the first line:

```javascript
const HER_NAME = "Beautiful";  // 👈 Change this to her actual name
```

That's it! The name will automatically appear throughout the website.

## 🚀 How to Use

### Option 1: Open Locally
1. Simply double-click `index.html` to open it in your browser
2. Share your screen or send her the files

### Option 2: Host Online (FREE)
1. Go to [GitHub Pages](https://pages.github.com/), [Netlify](https://www.netlify.com/), or [Vercel](https://vercel.com/)
2. Upload all files (index.html, styles.css, script.js)
3. Get your unique URL and share it with her!
